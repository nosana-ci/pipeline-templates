# Ornith

Ornith is a self-improving family of open-source models for agentic coding. Served through Ollama, this template deploys the open-weight `ornith:9b` model.

## Included Model

### 9B
- **Model Tag**: `ornith:9b`
- **Model Size**: 5.6 GB
- **Context Length**: 256K tokens
- **Modalities**: Text
- **VRAM Required**: ~7 GB
- **Use Case**: Lightweight agentic coding; low-VRAM performance baseline

## Highlights

- **Small footprint**: Runs on ~all benchmark-class GPUs
- **Long context**: 256K tokens
- **Tool ready**: Designed for coding assistants and agentic workflows
